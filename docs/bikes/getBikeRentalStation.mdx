# getBikeRentalStation

```typescript
(stationId: string) => Promise<BikeRentalStation>
```

`getBikeRentalStation` finds a single bike rental station by its ID.

## Parameters

### stationId (`string`)
The ID of the bike rental station you are interested in. The method will return a Promise which will resolve to an object of type [BikeRentalStation](flow-types/BikeRentalStation.js).

## Example

```javascript
import createEnturService, { convertFeatureToLocation } from '@entur/sdk'

const service = createEnturService({
    clientName: 'awesomecompany-awesomeapp'
})

async function example() {
    try {
        const bikeRentalStation = await service.getBikeRentalStation('YBE:VehicleSharingParkingArea:368')
        console.log(bikeRentalStation)
    } catch (error) {
        console.error(error)
    }
}
```
